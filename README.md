## 💻 How to Run This Project on Your Computer

If you want to download this project and see how it works on your own machine, follow these steps.

### What You Need First

You will need **Node.js** installed on your computer. You can download it for free here:

- [Download Node.js](https://nodejs.org/)

### Setup Steps

1. **Clone (download) this repository:**

   ```bash
   git clone [https://github.com/vbtrigona/vbtrigona.git](https://github.com/vbtrigona/vbtrigona.git)
   ```

2. **Open the project folder

    ```bash
    cd vbtrigona
    ```

3. **Install the project files**

    ```bash
    npm install
    ```

4. **Start the local website server:**

    ```bash
    npm run dev
    ```

5. **View the website**
    Open your internet browser and go to [http://localhost:3000](http://localhost:3000)
